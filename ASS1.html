WEB

A website can be written using three (3) languages :

  1. HTML
  2. CSS
  3. JavaScript

  • An Internet is a Computer Networking System used to connect to different users
  • The World Web / Web is a way of accessing information over the medium of the Internet
  • A browser is a program/software that allows you to interact and view contents of the web
  • Clients are devices (e.g Laptops, Tablets, Mobile Phones e.t.c) from which browsers are used
  • Servers is a dedicated machine used to serve out pages to client online 24/7

HOW DOES THE WEBSITE WORKS

  • Firstly, the clients sends URL(Uinform Resources Location) to ISP (Internet Service Provider)
  • ISP sends it to s a DNS (Domain Name System) Server

Note:. IP address is the protocol the internet uses for communication. The protocol for Website are "https" (hypertext transfer protocols). The First page of every website is the Homepage consisting of HTML, CSS, JavaScript.

HTML

  • HyperText: It helps us move around the page quickly. Examples are links you find on web pages, they link webs and pages together.
  • Mark-Up: This annotates the text and interprets it to the user for understanding.

TYPES OF MARK-UP LANGUAGE

  1. SGML : Standard General Mark-Up Language
  2. XML : Extensible Mark-Up Language
  3. HTML : HyperText Mark-Up Language

TYPES OF TAGS

  1. Container Tag
  2. Self Closing Tag
  3. Empty Tag
  4. Comment Tags are meant for Documentation []

Attributes: They are used to change the default appearance/settings of a tag. The syntax is "name="value". It can only be used within the opening tags.
